# Receitas Brasileiras

Site com receitas tradicionais brasileiras. Busca por nome, ingrediente ou categoria, com modo de preparo e lista de ingredientes.

## Receitas incluidas

Feijoada, Coxinha, Pao de Queijo, Brigadeiro, Moqueca, Empadao, Acai, Baiao de Dois, Pudim, Acaraje, Tapioca, Bolo de Cenoura, Arroz Carreteiro, Pastel, Misto Quente, Bolinho de Chuva, Frango a Parmegiana e Farofa.

## Tecnologias

- HTML
- CSS
- JavaScript

## Como usar

Abra o arquivo `index.html` no navegador.
